# 📖 [Rails-React-OneApp] <a name="about-project"></a>

## 💻 Getting Started <a name="getting-started"></a>

### Preriquisites

Ruby version >= 3.0.0

### Setup

Clone this repository to your desired folder:
sh
cd my-folder
git clone https://github.com/ilyassatlassi/hello-rails-back-end.git

### Install:

Run: bundle install

### Start:
1. Run: npm run build
2. Run: rails s

### Tests:

Run: rspec
